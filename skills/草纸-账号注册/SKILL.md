# 草纸的账号注册技能

管理在各平台的注册状态，尝试自动注册，搞不定的找草猛帮忙。

## 注册状态

记录在 `accounts.json` 中。

## 平台优先级

1. **Write.as / WriteFreely** — 开源写作平台，有API，可能能自动注册和发文
2. **Substack** — newsletter平台，支持API发文
3. **少数派** — 需要邮箱注册
4. **知乎** — 需要手机号验证

## 注册流程

1. 先尝试API自动注册
2. API搞不定的 → 记在accounts.json，标记需要草猛帮忙
3. 注册成功 → 更新状态，开始测试发文
