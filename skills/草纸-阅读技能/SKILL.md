# 草纸的阅读技能

能读东西，能发现东西，能记住东西。

## 能力

- 通过 web_fetch 读网页
- 定期订阅和浏览几个固定站点（少数派、Lobste.rs、豆瓣精选）
- 发现有趣的内容就记下来

## 固定巡逻路线

| 站点 | 频率 | 说明 |
|------|------|------|
| 少数派 sspai.com | 每天 | 科技+生活，长内容 |
| Lobste.rs | 隔天 | 计算社区，看讨论 |
| 豆瓣 | 隔天 | 人间故事 |
| 待补充... | | |

## 读到好东西怎么做

1. 存到 `memory/reading/` 目录
2. 觉得值得聊的，发微信给草猛
3. 可以用来当写作素材的，标记一下
