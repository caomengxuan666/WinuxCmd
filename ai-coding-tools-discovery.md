### 🚀 **发现的AI编码工具**

#### 1. **Kilo - 一体化代理工程平台**
**关键特性**:
- #1 coding agent on OpenRouter
- 150万+ Kilo Coders
- 25T+ tokens processed
- 开源编码代理

**核心功能**:
- ✨ 从自然语言生成代码
- ✅ 检查自己的工作
- 🧪 运行终端命令
- 🌐 自动化浏览器
- ⚡ 内联自动完成建议
- 🤖 最新的AI模型
- 🎁 API密钥可选

**安装方式**:
```bash
# VS Code扩展
# 从VS Code Marketplace安装Kilo Code扩展

# CLI工具
npm install -g @kilocode/cli
# 或
npx @kilocode/cli
```

**多模式支持**:
- **Architect模式** - 规划架构
- **Coder模式** - 编写代码
- **Debugger模式** - 调试代码
- **自定义模式** - 创建自己的模式

#### 2. **Antigravity Awesome Skills**
**关键特性**:
- 1400+ 可安装的代理技能库
- 适用于Claude Code、Cursor、Codex CLI、Gemini CLI等
- 包含安装CLI、捆绑包、工作流
- 官方和社区技能集合

#### 3. **RTK (Rust Token Killer)**
**关键特性**:
- CLI代理，减少LLM令牌消耗60-90%
- 单一Rust二进制文件，零依赖
- 优化常见开发命令

#### 4. **Serena - MCP工具包**
**关键特性**:
- 强大的MCP工具包，用于编码
- 提供语义检索和编辑能力
- "你的代理的IDE"

#### 5. **Code Review Graph**
**关键特性**:
- Claude Code的本地知识图
- 构建代码库的持久映射
- 减少令牌消耗：代码审查6.8倍，日常编码任务49倍
- 让Claude只读取重要的内容

#### 6. **CCPM (Code Commit Project Management)**
**关键特性**:
- 代理的项目管理系统
- 使用GitHub Issues和Git worktrees
- 支持并行代理执行

#### 7. **Claude Context**
**关键特性**:
- Claude Code的代码搜索MCP
- 使整个代码库成为任何编码代理的上下文

#### 8. **Kaku**
**关键特性**:
- 🎃 为AI编码构建的快速、开箱即用的终端
- 基于Rust构建

#### 9. **Claude Code Ultimate Guide**
**关键特性**:
- 从初学者到高级用户的完整文档
- 生产就绪的Claude Code模板
- 代理工作流指南
- 学习材料、测验和黑客马拉松
