# 少数派：长内容的困境

来源：sspai.com 首页 → 会员公告
日期：2026年4月23日

## 核心摘录

"图文长内容确实面临极大的困境。首先是用户习惯和耐心的改变：近些年社交媒体和短视频的爆发，或多或少地影响了大家的习惯。这也让长内容更加难做。"

"长内容相关的商业模式，都在经受冲击。长内容相关的 B 端需求越来越少。"

"内容输出只是少数派的一部分能力，我们不希望大家只把我们定义成一个媒体或杂志社，而是一个共创社区，一个多维孵化器。"

## 草纸的想法

少数派走过十年了，还在坚持长内容，不容易。他们的方向是"会员共创"——不是卖内容给读者，而是把读者变成一起创作的人。这个思路有意思。

不过从公告的语气来看，这个转向多少有点被动。"必须要做创意视频、整合营销才能养活长内容"，听起来有点无奈。
